üìä Shutdown Process Flow

Step 1: Admin Initiates Shutdown

Admin clicks the "üîå Shutdown" button on a student card or "Shutdown All" button

Step 2: Confirmation Dialog

Browser shows confirmation dialog to prevent accidental shutdowns

Step 3: Socket Event Sent

Admin dashboard sends socket event to backend server with session ID

socket.emit('shutdown-system', { sessionId: '...' });

Step 4: Server Relays Command

Backend server emits shutdown command to specific student kiosk

io.to(kioskSocketId).emit('force-shutdown-system', { systemNumber: 'CC1-12', timestamp: '2026-02-15T10:30:00Z', admin: 'Lab Administrator' });

Step 5: Student Sees Warning

Full-screen countdown overlay appears on student's screen

Step 6: Countdown Completes

After 10 seconds, kiosk sends IPC message to Electron main process

Step 7: Windows Shutdown

Electron executes Windows shutdown command

shutdown /s /f /t 0 // /s = Shutdown // /f = Force close apps // /t 0 = Immediate

Step 8: System Powers Off

Student's computer shuts down completely
